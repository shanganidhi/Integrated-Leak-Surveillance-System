# Mobile App Architecture

The mobile app provides real-time visibility into the gas monitoring system.

## Tech Stack
- **Framework:** Flutter (Recommended) or React Native.
- **Charts:** `fl_chart` or `victory_native`.
- **API Communication:** `http` or `axios`.

## Key Components
1. **Live Dashboard:** Real-time PPM and Rs/R0 graphs.
2. **Alert Manager:** Push notifications for anomaly thresholds.
3. **Data History:** Heatmaps of gas concentration over time.
4. **Calibration Tool:** Manual R0 override and baseline update trigger.
